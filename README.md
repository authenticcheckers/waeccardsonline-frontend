# WAECCARDSONLINE - Frontend bundle

This is the complete frontend ready for Vercel deployment.

- Put the contents of this folder at the root of your GitHub repo connected to Vercel.
- Replace the Paystack public key in `index.html` where it says `REPLACE_WITH_PAYSTACK_PUBLIC_KEY`.
- Ensure admin files are in `/admin` (admin.html, admin.css, admin.js).
- Backend URL is configured to: https://waecghcardsonline-backend.onrender.com

Deploy: commit & push -> Vercel auto-deploys.
